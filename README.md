# ERP Data Audit - SQL Server

Projeto simulado baseado em cenários reais de suporte técnico e auditoria de dados em sistemas ERP educacionais.

## Objetivo
Demonstrar técnicas de análise de dados, troubleshooting e validação de regras de negócio utilizando SQL Server (T-SQL), com foco em identificação de inconsistências acadêmicas e financeiras.

## Contexto
O projeto simula um ambiente acadêmico contendo dados de alunos, matrículas e financeiro, permitindo reproduzir cenários comuns de incidentes em sistemas corporativos, como divergência de status, ausência de vínculos e falhas de integração.

## Principais Atividades
- Auditoria de dados acadêmicos e financeiros
- Identificação de inconsistências de negócio
- Validação de regras utilizando SQL
- Análise e reprodução de incidentes
- Criação de queries para troubleshooting

## Tecnologias
- SQL Server
- T-SQL

## Estrutura do Projeto
- `database/` → scripts de criação e carga de dados  
- `queries/` → consultas de auditoria e validação  
- `docs/` → documentação e casos de teste  

## Como Executar
1. Executar o script `database/schema.sql` para criação das tabelas  
2. Executar o script `database/seed.sql` para carga de dados  
3. Executar as queries em `queries/` para validação das regras de negócio  

## Observações
Este é um projeto fictício, desenvolvido para fins de estudo e demonstração técnica, sem utilização de dados reais ou informações confidenciais.
