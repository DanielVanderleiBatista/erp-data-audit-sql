# ERP Data Audit - SQL Server

Projeto simulado baseado em cenários reais de suporte e auditoria em sistemas ERP educacionais.

## Objetivo
Demonstrar técnicas de análise de dados, troubleshooting e validação de regras de negócio utilizando SQL Server.

## Contexto
O projeto simula um ambiente acadêmico com dados de alunos, matrículas e financeiro, permitindo identificar inconsistências comuns em sistemas corporativos.

## Principais Atividades
- Auditoria de dados acadêmicos e financeiros
- Identificação de inconsistências
- Validação de regras de negócio
- Suporte à análise de incidentes
- Criação de queries para troubleshooting

## Tecnologias
- SQL Server
- T-SQL

## Estrutura
- database: scripts de criação e carga de dados
- queries: consultas de auditoria e validação
- docs: documentação e casos de teste

## Observação
Projeto fictício, sem uso de dados reais ou informações confidenciais.
